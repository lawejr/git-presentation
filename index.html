Немного терминологии

  • GIT - распределенная система контроля версий
  • Repository - все файлы проекта. Может быть локальным и удаленным.
  • Branch - ветка в проекте
  • Commit - фиксация сделанных изменений
  • Push - отправка локальных изменений в удаленный репозиторий / гл. Запушить/Залить
  • Pull - скачать изменения из удаленного репозитория / гл. Запулить/Стянуть
